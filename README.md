Код который выводит на экран:
– среднюю букву, если число букв в слове нечётное;
– две средних буквы, если число букв чётное

word = input("Введите слово: ").strip() # Вводим слова и удаляем пробелы из него

length = len(word) # Определяем количество символов в слове

index_middle = length // 2 #Вычисляем индекс середины слова 

# Задаем условие и определяем результат
result = word[index_middle-1:index_middle+1] if length % 2 == 0 else word[index_middle] 

print(f"Результат: {result}") # Выводим результат

Код для сортировки имен по алфавиту с целью познакомить людей с одинаковыми индексами после сортировки.
# Списки имен
boys = ['Peter', 'Alex', 'John', 'Arthur', 'Richard']
girls = ['Kate', 'Liza', 'Kira', 'Emma', 'Trisha']

#Основной цикл
if len(boys) == len(girls): # сравниваем количество симфолов в именах
    boys.sort() # сортируем список юношей по алфовиту
    girls.sort() # сортируем список девушек по алфовиту
    print('Идеальные пары:')
    for sort_boys, sort_girls in zip(boys, girls): # объединяем отсортированные имена в пары
        print(f'{sort_boys} и {sort_girls}') 
else:
    print('Внимание, кто-то может остаться без пары!')
